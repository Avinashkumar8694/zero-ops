# Sub Process

Embedded process grouping node.

## What This Node Does
Represents a contained subprocess inside the same diagram. Use it to visually group a section of the process into a smaller scope.

## How To Use It
- Use it when a portion of the flow belongs together conceptually.
- Use call activity instead when the logic should be reusable across assets.

## Validation Notes
- This node currently has no custom properties.
- Embedded subprocess runtime behavior will depend on the surrounding engine implementation.
